## 技术架构特点

### 1. 统一技术栈
- **Web框架**: FastAPI + Uvicorn
- **数据验证**: Pydantic v2
- **测试框架**: pytest + httpx
- **CLI工具**: Typer
- **代码质量**: Black + isort + mypy

### 2. 模块化设计
```
packages/backend-python/
├── src/
│   ├── api/           # API路由层
│   ├── core/          # 核心配置
│   ├── database/      # 数据库连接
│   ├── schemas/       # Pydantic模式
│   ├── services/      # 业务逻辑层
│   └── main.py        # FastAPI应用
├── tests/             # 测试代码
├── main.py            # CLI入口
└── pyproject.toml     # 项目配置
```

### 3. 类型安全
- 使用Pydantic进行数据验证
- 严格的类型注解
- 自动API文档生成
